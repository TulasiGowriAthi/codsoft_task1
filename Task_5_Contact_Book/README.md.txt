# Task 5: Executive Contact Directory (Contact Book)

A complete graphical CRUD (Create, Read, Update, Delete) directory software built to organize, search, and modify profile cards.

## Features
- **Complete Profile Cards:** Tracks Name, Phone Number, Email, and Addresses seamlessly.
- **Real-Time Search Bar:** Dynamic list filtration that matches string entries as you type.
- **Persistent Data Store:** Saves profiles directly into `contacts.json`.
- **Integrated Control Panel:** Simple operations to input new records, edit active profiles, or wipe target rows instantly.

## Tech Stack
- **Language:** Python 3
- **GUI Framework:** CustomTkinter
- **Data Architecture:** JSON Engine
