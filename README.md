# Task 1: Premium Task Manager (To-Do List)

A modern, sleek desktop application designed to track tasks, built with a dark-themed user interface and local data persistence.

## Features
- **Modern Dark-Mode UI:** Crafted using `CustomTkinter` for a premium look.
- **Data Persistence:** Tasks are written to and loaded from a local `tasks.json` file automatically, preventing data loss on application close.
- **Dynamic Checkboxes:** Interactive task completion toggles that gray out finished tasks.
- **Input Validation:** Warning pop-ups to handle and prevent accidental empty entries.

## Tech Stack
- **Language:** Python 3
- **GUI Framework:** CustomTkinter
- **Storage Profile:** JSON Engine
