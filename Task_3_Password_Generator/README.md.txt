# Task 3: Secure Password Forge (Generator)

A cryptographically secure desktop application that dynamically creates highly customizable random passwords based on length and complexity.

## Features
- **Interactive Length Slider:** Seamlessly adjust password length constraints from 8 to 32 characters.
- **Complexity Toggles:** Checkboxes to explicitly include/exclude uppercase letters, numbers, and symbols.
- **Advanced Cryptography:** Engineered utilizing Python's native `secrets` module for high-security production readiness.
- **Clipboard Utility:** Copies the generated strings instantly to your system clipboard for easy deployment.

## Tech Stack
- **Language:** Python 3
- **GUI Framework:** CustomTkinter
- **Core Standard Libraries:** `secrets`, `string`
